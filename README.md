PROGRAM PURPOSE:
The purpose of this POS system is to reduce the complexiitty of a cashier's everyday work life 
by introducing a simplified interace meant to allow cashiers to add items to a cart, 
remove items, view the cart, calculate the total bill accept payment, 
and generate a receipt & ultimately reduce human errors.
|||
HOW TO RUN PROGRAM:
Download pycharm
Download the file Hall.Nathan-POS-ITT103-SP2026.py
Open The prompt
Start using the POS system
|||
ASSUMPTIONS/LIMITATIONS:
The system doesn't have a clerk system to identify who's using it.
It doesn't have a security feature, to protect the business & cashier's.
The system can only accept jamaican currency from customers.
The system doesn't need WIFI to operate.
The system can't accept credit/debit card payments
